//******************************************************************************
// Server Installatie Instructies
//******************************************************************************


1. 	Op de server, maak een nieuwe map in de map Mods (die zich bevindt in de CoD4 root) en noem deze pam4.
2. 	Upload de .iwd en .ff bestanden naar de nieuwe map (pam4) op de server.
3. 	Zet de parameter fs_game in de commandline van je server naar "mods/pam4" (dus +set fs_game mods/pam4)
4. 	Upload alle .cfg files ook naar mods/pam4. Pas deze aan aan je wensen (let op de onderste regel!) en 
	zorg dat er in de commandline staat: +exec server.cfg

5. 	Restart de server.
